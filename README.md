# 🎉 春节主题编程动画/小游戏

两个有趣的春节主题前端互动作品，使用纯HTML5、CSS3和JavaScript实现。

## 🎮 游戏介绍

### 1. 🎆 春节烟花模拟器
- **描述**: 点击屏幕发射绚丽的春节烟花，体验烟花的爆炸效果和声音
- **功能**:
  - 点击按钮发射不同颜色的烟花（红、金、绿、蓝）
  - 在画布上点击发射随机烟花
  - 烟花自动爆炸并产生粒子效果
  - 不同颜色的烟花有不同的爆炸效果和声音
  - 清除所有烟花功能

### 2. 🧧 接红包大作战
- **描述**: 控制篮子接住从天上掉落的红包，避免接到鞭炮
- **功能**:
  - 使用鼠标控制篮子左右移动
  - 接住红包获得10分
  - 接到鞭炮失去1点生命
  - 每接住10个红包获得额外生命
  - 随着时间增加游戏难度
  - 游戏结束显示最终得分

## 🚀 快速开始

1. 克隆或下载项目文件
2. 直接在浏览器中打开 `index.html` 文件
3. 无需安装任何依赖，所有代码都在本地运行

## 🎯 游戏控制

### 烟花模拟器
- **点击画布**: 在点击位置发射随机烟花
- **清除按钮**: 清除所有烟花效果

### 接红包游戏
- **鼠标移动**: 控制篮子左右移动
- **开始游戏**: 开始游戏
- **暂停游戏**: 暂停游戏
- **重新开始**: 重置游戏并重新开始

## 🛠️ 技术特点

- **纯前端实现**: 使用HTML5 Canvas绘制图形
- **响应式设计**: 适配不同屏幕尺寸
- **音频效果**: 使用Web Audio API实现音效
- **粒子系统**: 实现烟花爆炸和粒子效果
- **碰撞检测**: 精确的矩形碰撞检测算法
- **游戏循环**: 使用requestAnimationFrame实现流畅动画
